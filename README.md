# Java API - Accounts

API REST simples para gerenciamento de contas bancárias, desenvolvida em Java com Spring Boot.

## Funcionalidades
- Criar conta
- Listar contas
- Buscar conta por ID
- Depositar valor
- Sacar valor (com validações)

## Tecnologias
- Java
- Spring Boot
- Maven

## Endpoints
- POST /accounts
- GET /accounts
- GET /accounts/{id}
- POST /accounts/{id}/deposit
- POST /accounts/{id}/withdraw
